## 📋 What is this application?

The UTS_Praktikum_Mobile application helps users organize mobile project data on a Windows computer. It provides a clean interface to track progress and handle files. ## 💡 How to use the app

Open the application by clicking the desktop icon. The main window shows your dashboard. Use the navigation menu on the left to switch between different data views. 

### Importing project files
You can bring your existing mobile project files into the app. Select File from the top menu, then choose Import. Select the folder containing your data. The application reads your files and displays them in the list view.

### Sorting your tasks
The app sorts projects by date by default. Click the column headers to change the order. You can sort by name, priority, or status.

### Saving your progress
The application saves your work as you make changes. You do not need to click a save button. If you want to create a manual backup, select File, then Export. The app creates a secure file you can store on a thumb drive or cloud storage.
